Woobie Module
-------------

This is for demonstrative purposes only. If you are looking for starting
point in writing modules, woobie is the right thing.

This module requires: none

**Configuration** — add to your ``eggdrop.toml``::

  [modules]
  load = [
    ...
    "woobie",
  ]
